PomeGraph – Graphical Tool to bind Epidemiological and Genome information

About

Pomegraph is a graphical tool for visualization and handling the contact trace diagram especially for SARS-CoV-2 analysis. This software was developed upon iGraph.js, a demo application of interactive graph visualization. The Pomegraph draws mutual linkage among subjects based on epidemiological and genomic reconessence information, that is represented as labeled nodes which are colored by epidemiological group. The original ideas behind these interaction techniques have been published in:

  • Tominski, C.; Abello, J. & Schumann, H.: CGV - An Interactive Graph Visualization System. Computers & Graphics, Vol. 33, No. 6, 2009.
  • Tominski, C.; Abello, J.; van Ham, F. & Schumann, H.: Fisheye Tree Views and Lenses for Graph Visualization. Proc. of IV, London, IEEE, 2006.

How to Use

This application have two tab panels. After loading input data (JSON format), check the basic physical (epidemiological) association of subjects by "Field Epi (Epidemiolog View)" tab first. Then, switch to "Genome Epi (Genome View)" to analyze composite information.
Click tagged shapes to show detailed information of individuals on right upper of the window. On "Genome Epi", epidemiological links and genomic links are shown in blue and red lines respectively.

Data Input

An epidemiological information (physical data, .xlsx format) is required to draw a view. Optionally, you can combine the genomic information by the result data of pylogenetic analysis (phylogenetic data:[Newick format]) files in file selector boxes below. Combined result will be shown in "Genome Epi".

Local Zoom Function

Zoom and pan can be controlled via wheel, right drag, and double click.

Finally, if you are not satisfied with the generated layout, you can move nodes by dragging mouse and stick the position by release with pressing shift key. Should you get lost, press to reset the view. If you prefer a full-page view, press to open it in new window and check it out!
